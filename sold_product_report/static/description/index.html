Community Enterprise

Features

  • This module helps user to print report of sold items/products for specific time or specific categories.
  • It helps user to measure the sales report between specific time.

Configuration

  • For configuration user has to only install the Product sales report app.

Report Type

  • By selecting the sale as report type, user can get the sold items/products report between entered time.
  • If user want to print report of sold products from POS (Point of sale) then user has to select the Point of Sale as report type.
  • By selecting the both as report type, user can get report of sold items from sale and POS both.


Product Category

  • Here user can also enter the product categories for only print the report of specific category.

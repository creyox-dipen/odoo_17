Auto Add Followers via Mail Alias

Watch Video

Overview

The Auto Add Followers via Mail Alias module is a powerful email routing extension that seamlessly hooks into Odoo's core mail thread system. It entirely automates the process of managing record followers by analyzing incoming email headers.

In the standard Odoo flow, when an email is received, the system checks the To and CC fields. If a contact with the same email address already exists, that contact is added as a follower. However, if no matching contact is found, Odoo simply skips that email address and does not add it as a follower.

Our module enhances this behavior. If any email address in the To or CC headers does not exist as a contact, the module automatically creates a new contact using that email address and then adds the newly created contact as a follower to the record.

Features

Reads and processes all recipients from both To and CC fields of incoming emails.
Automatically creates a new contact if an email address from the To or CC field does not already exist in the system.
Eliminates the need to manually create contacts or add followers after receiving emails.
Extends the standard mail alias flow without disrupting the default behavior.
Ensures all relevant email recipients stay linked to the correct record.
